 # 🌐 Huddle Landing Page

Landing page responsiva desenvolvida com **HTML5** e **CSS3 (Flexbox)**, baseada no desafio **[Frontend Mentor](https://www.frontendmentor.io/profile/AnaCCapel)** Huddle landing page with a single introductory section.

## 🎯 Objetivo:
